# Water-Works-Company-Trouble-Ticket-Management-System
Water Works Company Trouble Ticket Management System is a project which is used by customers of WWC for resolving failures. Thus, the overall system will consist of WWTTMS Fault Tracking System and WWTTMS Web Interface. The WWTTMS Fault Tracking System will work with the web interface, it takes the type of fault that users enter and provides a return to the technical team. WWTTMS provides a more efficient communication between the company and the customers and creates a faster solution.This system will not only solve the malfunction, but also send information to customers after the malfunction.
The objectives of the project are:
-Getting the necessary operations by getting enough explanation from the subscriber about the malfunction
-Saving the fault information from the subscriber to the system
-For the location of the malfunction information, the technical chief assigns the appropriate team to that location.
-Technical team to correct the related fault.
-E-mail sending to the subscriber as a return after the fault is resolved.
